# Dicas do terminal
vou listar alguns comandos que estamos aprendendo:

- `cd` (navega entre pastas)

Exemplo para entrar nas pastas:
```
cd nome-da-pasta
```

Exemplo para sair de uma pasta:
```
cd ..
```
- `mkdir` (cria pastas)
Exemplo de como criar pastas
```
mkdir nome-da-pasta
```
